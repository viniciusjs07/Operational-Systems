Exemplo de execução de run_lab4.sh:

bash run_lab4.sh 1 > lab4_1.txt ou 
bash run_lab4.sh 2 > lab4_2.txt onde:

a argumento 1 equivale a política FIFO e o argumento 2 
equivale a política RR.

Após a execução acima, é necessário realizar o tratamento dos dados para deixar no formato do saidalab4.txt que será utilizado no script de geração dos plots.

bash plot_lab4.sh
